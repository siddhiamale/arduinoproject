Automatic Motor Service

How it works:-

It can be designed for various applications such as automatic water pumps,smart irrigation systems, or motor contol in industries. Below is a general working principle of particular sensors which are used name in this Automatic motor serives project using Arduino.

1. L298N motor:- It works using H-Bridge circuits, which allows motor to spin forward, backward and stop the motor. If 1st input is high and 2nd input is low then motor moves forward. If 1st input is low and 2nd is high then motor moves backward. If both inputs are low then motor stops or both inputs are high then motor breaks otherwise stops permanently.

2. Ultrasonic Sensor:- It works for object detection, distance-based motor controlling based on distance measurement by formula. Formula:- It uses for measure the distance of water detection for stop the DC motor automatically.

3. DC Motor:- It control using L298N motor driver and Ultrasonic sensor when the Arduino can supply enough directly to the motor. Depending on the signals received, the motor moves as per our requirement.

3. DC Motor:- It control using L298N motor driver and Ultrasonic sensor when the Arduino can supply enough directly to the motor. Depending on the signals received, the motor moves as per our requirement.

4. Arduino UNO Board:- It acts as brain of the systems, controlling the motor based on sensors inputs, user commands or pre-programmed logic . Based on the received data, the Arduino decides weather to start, stop, change direction or adjust the speed of the motor also it can sends the signal to motor driver to control the system.

Componants needed:-

1. Arduino UNO

2. L298N motor driver

3. Ultrasonic Sensor

4. 5v battery

5. External power supply(9v)

6. Jumper cabels (M-M,M-F,F-F)

Algorithm:-

Step 1: Start.

Step 2: Initialize Arduino and components: Set up motor driver pins as OUTPUT. Set up ultrasonic sensor pins as INPUT/OUTPUT.

Step 3: Measure distance using the ultrasonic sensor: Sends a trigger pulse. Receive echo pulse and calculate the distance.

Step 4: Check distance conditions: If distance > 10cm, stops motor automatically. If 1st motor stops, then it stops permanently’ If distance <= 20cm, starts motor automatically.

Step 5: Repeat steps continuously.

Step 6: Stop.
